# Two Dads Having a Beer — GitHub Pages Starter

This is a static starter site for GitHub Pages.

## Files
- `index.html` — homepage
- `style.css` — site styling
- `script.js` — simple mobile menu + footer year
- `404.html` — custom not found page

## Customize first
- Replace `youremail@example.com` in `index.html`
- Update episode links
- Update sponsor copy
- Add your logo later if you want
